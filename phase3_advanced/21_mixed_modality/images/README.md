# 图片目录

请在此目录下放置测试图片。

## 所需图片

| 文件名 | 用途 | 说明 |
|--------|------|------|
| sample.jpg | 基本图像描述 | 任意图片即可 |
| text_image.jpg | OCR文字识别 | 包含文字的图片 |
| chart.png | 图表分析 | 柱状图、折线图等 |
| image1.jpg | 多图对比 | 用于对比的第一张图 |
| image2.jpg | 多图对比 | 用于对比的第二张图 |

## 示例图片来源

你可以从以下网站获取免费图片：
- https://unsplash.com/ (高质量免费照片)
- https://pixabay.com/ (免费图片和视频)
- https://pexels.com/ (免费图片)

或者直接使用你自己的图片。

## 注意事项

- 支持的格式：jpg, jpeg, png, gif, webp
- 建议图片大小不超过 5MB
- 如果示例提示图片不存在，请确保文件名正确
