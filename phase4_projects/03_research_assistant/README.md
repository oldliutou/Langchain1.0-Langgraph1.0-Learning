# 项目三：AI 研究助手

## 📋 项目概述

本项目构建一个功能完整的 AI 研究助手，能够帮助用户进行信息收集、文献分析、内容总结和报告生成。系统整合了多种 AI 能力，是一个实用的生产力工具。

## 🎯 项目目标

通过本项目，你将学会：
- 构建多步骤研究工作流
- 实现智能信息提取和总结
- 设计结构化输出格式
- 整合多种数据源
- 实现迭代式内容优化
- 构建交互式研究界面


## 🔧 核心功能

### 1. 研究规划
- 主题分析和分解
- 研究问题生成
- 研究计划制定

### 2. 信息收集
- 模拟网络搜索
- 知识库查询
- 信息验证

### 3. 分析综合
- 多源信息整合
- 观点对比分析
- 关键发现提取

### 4. 报告生成
- 多种报告格式
- 结构化输出
- Markdown 导出

## 📊 输出格式

### 简报 (Brief)
- 执行摘要
- 关键发现
- 建议行动

### 详细报告 (Detailed)
- 完整背景
- 详细分析
- 数据支持
- 结论建议

### 学术报告 (Academic)
- 摘要
- 引言
- 方法论
- 结果讨论
- 参考文献

## 🚀 快速开始

```bash
# 1. 设置环境变量
export OPENAI_API_KEY="your-api-key"

# 2. 运行研究助手
python main.py
```

## 💡 使用示例

```python
from main import ResearchAssistant

# 创建研究助手
assistant = ResearchAssistant()

# 执行研究
result = assistant.research(
    topic="人工智能在医疗领域的应用",
    depth="detailed",
    output_format="markdown"
)

# 获取报告
print(result["report"])
```
